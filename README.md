## ⚙️ Installation & Running Locally

1. **Clone** the repo
```bash
   git clone https://github.com/yourusername/portfolio.git
   cd portfolio
   Install dependencies
   npm install
   Run in development mode
   npm run Dev
```
Open http://localhost:3000 and explore!
